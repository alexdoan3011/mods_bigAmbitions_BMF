# Better Map Filter

A small mod for [Big Ambitions](https://store.steampowered.com/app/1331550/Big_Ambitions/).

The vanilla City Map filter is one long vertical list of toggle switches you have to scroll forever to find anything. This mod turns it into a 3-column grid of square buttons.

## Install

Subscribe to it on the Steam Workshop.

## Build from source

This is a standard Big Ambitions SDK mod. Drop the `BetterMapFilter` folder into `Assets/Mods/` in the [modding SDK project](https://github.com/hovgaardgames/bigambitions), then use **Big Ambitions > Mod Builder > Build & Install**.

See [DEVELOPER_NOTES.md](DEVELOPER_NOTES.md) for how it works under the hood.
